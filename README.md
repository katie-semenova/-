# Прогноз цен на удобрения (Карбамид, DAP, MOP)

Этот проект представляет собой решение для прогнозирования цен на три ключевых вида минеральных удобрений: Карбамид, DAP (диаммоний фосфат) и MOP (хлористый калий). Основная цель — предоставить коммерческому отделу числовой ориентир для принятия решений о заключении контрактов.

## 1. Бизнес-задача

Коммерческая команда ежедневно сталкивается с двумя потоками информации: историей цен и рыночными новостями. Связать их вручную для принятия решений ("фиксировать цену сейчас или подождать?") сложно, и менеджеры часто полагаются только на интуицию.

Цель проекта: Создать модель, которая на основе исторических данных и новостного фона дает числовой прогноз цены на 4 недели вперед.

Критерии успеха:

Точность: Прогноз должен быть точнее простого предположения «цена не изменится».
Практическая польза: Ошибка прогноза (MAPE) должна находиться в пределах 5–10%.
Доверие: Результаты должны быть интерпретируемыми и обоснованными для бизнес-пользователей.

## 2. Концепция решения

Выбор подхода: Вместо сложного анализа новостей (который дает контекст, но не цифры), я сосредоточилась на числовом прогнозе цены. Это дает прямой ответ на главный бизнес-вопрос: "Какой будет цена через месяц?".

Выбор модели: использую LightGBM.

Почему не нейросети: всего ~339 недель данных, что недостаточно для сложных нейросетей (LSTM/Transformer), которые склонны к переобучению на малых выборках.
Почему LightGBM: Эта модель хорошо работает с табличными данными, устойчива к выбросам и, что критически важно, позволяет интерпретировать результаты, показывая, какие факторы больше всего влияют на прогноз.

Горизонт прогноза (4 недели):

1 неделя: Слишком короткий срок для управленческого решения.
8+ недель: Слишком высокая неопределенность.
4 недели: Оптимальный срок, соответствующий типичному циклу согласования поставки.

## 3. Как это работает (Технические детали)

Данные:

Числовые признаки: Исторические цены на продукты (лаги), цены на внешние факторы (газ, аммиак, нефть, курс рубля, фрахт и др.), скользящие средние и сезонность.
Текстовые признаки (Новости): протестировала использование новостей (словарный сентимент, TF-IDF + SVD). На текущем объеме данных (370 заголовков) текст не дал значимого прироста точности. Однако для интерпретируемости в финальной модели оставлен признак "категориальный сентимент" (тональность новостей по ключевым темам, например, "Аммиак").
Модель: Для каждого из трех продуктов была обучена отдельная модель LightGBM с индивидуальным подбором гиперпараметров.

## 4. Результаты

Модель показала улучшение по сравнению с "наивным" прогнозом (предположением, что цена не изменится).

Продукт	   MAE, $/т	 MAPE, %	Naive MAE	 Улучшение, %
Карбамид	 15.19	   3.88	    24.52	     38.06
DAP	       24.22	   3.96	    31.32	     22.67
MOP	       29.16	   6.00	    31.23	     6.64
Вывод: Прогнозы по карбамиду и DAP значительно точнее наивного подхода. Прогноз по MOP улучшается скромнее, что объясняется его исторической стабильностью.

Текущий прогноз (на 4 недели вперед):

Карбамид: ~$400 → **~$405** (+1.4%)
DAP: ~$618 → **~$612** (-1.0%)
MOP: ~$489 → **~$494** (+1.1%)

## 5. Ограничения и будущие улучшения

Важно понимать границы применимости модели

Прогноз — не истина: Это числовой ориентир, а не точная цена. Модель дает точечный прогноз без доверительного интервала.
Новости: На текущем объеме данных их влияние на точность незначительно. Основная ценность новостей сейчас — в предоставлении контекста.
Непредсказуемые шоки: Модель не может спрогнозировать результаты санкций, форс-мажоров или резких изменений экспортной политики.
Независимые модели: Три модели обучаются независимо, что не позволяет учитывать корреляцию ошибок между рынками.

Планы по улучшению

Накопление данных: Сбор большего объема новостных данных для выделения полезного сигнала.
Квантильная регрессия: Переход к прогнозированию не только точечной цены, но и диапазона (например, "цена будет в вилке $400–420").
Учет корреляций: Разработка многомерной модели, которая учитывает взаимосвязи между рынками удобрений.
